# ⚓ 忒修斯之船 (Ship of Theseus) — 可视化交互系统

> **文档版本**: v1.1  
> **当前日期**: 2026-06-27  
> **系统隐喻**: 船不是木板。船是船上的人对彼此长久的呼唤与回应。只要呼唤与回应还在，木板换过一千次，它依然是同一艘船。

---

## 🌌 一、 产品概念与愿景

本项目旨在打破传统 AI 单调的“一问一答”命令行美学困境，利用 **React 19 + Pixi.js v8** 游戏级渲染引擎，在浏览器中构建一个由 **8个具备独立认知边界、记忆隔离与关系网络的 AI Persona** 组成的动态共生系统。

系统的视觉语言深度向 **《Persona 5 Royal》(P5R)** 的高张力波普美学致敬。通过斜角切割、粗黑描边、半色调网点、以及主循环级别的流式渲染，为用户打造一个地下、叛逆、却又不失温暖的思维研讨空间。

## 🏴‍☠️ 二、 视觉灵魂：红黑安那其主义 (Anarchism) 配色

本系统的核心视觉打破了传统商业软件的温和死板，采用了极具反叛色彩的**安那其主义（红+黑）**作为主色调：

- **核心主题色（红 #D40000 + 黑 #1A1A1A）**：此配色作为**用户（Lau）的主题色**贯穿全局。它代表着绝对的独立、去中心化的自由思考、以及对黑盒命令行美学困境的彻底叛逆。

## 🖥️ 三、 核心场景交互规划 (7大场景)

1. **场景1: 船全景 (Ship Panorama)**：启动画面。Pixi.js 多层视差动态场景，八个角色分布在甲板各处，点击进入交互。
2. **场景2: 1v1 对话 (Dialogue)**：单角色聊天。P5R 风左右斜切分屏，含角色立绘表情差分与打字机吐字动态。
3. **场景3: 研讨室 (Seminar Room)**：多人环形布局，处理复杂的串行发言队列，支持 Socrates 随时闪烁“闯入”。
4. **场景4: 档案室 (Archive)**：仿古籍排版，用于复盘历史研讨会内容。
5. **场景5: 心理咨询室 (Memory Ward)**：**宪法级记忆硬隔离查看器**，非当前激活角色严禁翻阅。
6. **场景6: 船长室 (Captain's Cabin)**：用户（Lau）的个人计划看板，承载角色异步留下的提示便签。
7. **场景7: 设置/系统 (Settings)**：纯本地安全托管 DeepSeek API Key 及其它全局配置。

## 📁 四、 核心目录结构说明

本项目底层全面接入高性价比的 **DeepSeek API**，利用原生的 `fetch` 与 `ReadableStream` 搞定毫秒级的流式响应，摆脱账单焦虑。通过引入 `useChatManager` 控制层，实现控制逻辑与 UI 渲染的完全解耦。
